# Meridian

Meridian is an open-source browser dual-pole puzzle-platformer about two mirrored halves sharing one sun at the solstice.

## Live Demo

GitHub Pages: https://<user>.github.io/meridian/

## Development

```bash
cd meridian
npm install
npm run dev
```

## Build

```bash
cd meridian
npm run typecheck
npm run check:determinism
npm run build
```

## Music

The project keeps `meridian/public/music/.gitkeep` tracked as a placeholder. Non-CC0 local music files placed in that directory are ignored by git.
